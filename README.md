Lista de Tarefas Android

Este aplicativo Android permite que os usuários gerenciem suas tarefas diárias de forma simples e eficiente. Ele oferece uma interface intuitiva para adicionar, remover e marcar tarefas como concluídas.
Funcionalidades Principais

    Adicionar Tarefas: Permite que os usuários criem novas tarefas com facilidade.

    Remover Tarefas: Opção para excluir tarefas que não são mais necessárias.

    Marcar como Concluídas: Funcionalidade para marcar tarefas como concluídas.

    Visualização de Tarefas: Tela principal que lista todas as tarefas atuais.

Arquitetura

O aplicativo foi desenvolvido utilizando a linguagem Kotlin, com foco em simplicidade e facilidade de uso. A estrutura do projeto segue boas práticas de desenvolvimento Android.
Tecnologias Utilizadas

    Kotlin: Linguagem de programação utilizada para desenvolver o aplicativo.

    Android SDK: Utilizado para criar a interface do usuário e gerenciar as funcionalidades do sistema operacional Android.

    bash
